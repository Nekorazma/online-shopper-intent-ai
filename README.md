Neuromarketing Purchase Predictor

This project is a research prototype designed to predict online shoppers' purchase intention using machine learning.

Overview

* **Model:** Random Forest Classifier
* **Accuracy:** 89%
* **Key Features:** PageValues, ExitRates, ProductRelated_Duration.

File structure:
- `online-shoppers-behavior-prediction.ipynb`: Exploratory Data Analysis (EDA) and business insights.
- `script.py`: Production-ready script for training and saving the model.
- `neuro_marketing_model.pkl`: The serialized AI model for deployment.
- `online_shoppers_intention.csv`: Dataset of 12,330 sessions.
